    Документация проекта: Cloude Ai(Генерация изображений с помощью нейросети)

    Описание проекта

    Данный проект представляет собой веб-сайт, на котором пользователи могут генерировать изображения с использованием нейросети. Сайт включает в себя систему аккаунтов, позволяющую пользователям управлять своими сгенерированными изображениями, а также функцию ленты, на которой отображаются изображения, созданные другими пользователями.

    Основные функции проекта

      • Пользователи могут вводить текстовые запросы (промпты) для генерации изображений.

      • Запросы отправляются на API нейросети, а результаты обрабатываются и отображаются на сайте.

      • Система аккаунтов

      • Пользователи могут создавать учетные записи для управления своими сгенерированными изображениями.

      • Реализована система аутентификации и управления сессиями, что обеспечивает безопасность учетных записей.

      Галерея пользователя

      • Каждому пользователю доступна персональная галерея, в которую сохраняются все сгенерированные ими изображения.

      • Пользователи могут просматривать, удалять или делиться своими изображениями.

      Общая лента

      • На главной странице отображается лента с изображениями, созданными всеми пользователями.

      • Каждое изображение сопровождается текстом промпта и другой информацией, что позволяет пользователям изучать работы друг друга.

      Навигация

      • Удобная навигация между различными разделами сайта (галерея, лента, профиль пользователя).

      • Интуитивно понятный интерфейс, позволяющий легко находить нужные функции.

      Технические детали

      Cтек Технологий

      • HTML & CSS & SCSS: для разметки и стилизации страниц.

      • JavaScript: для динамического взаимодействия с пользователем(логика всего сайта)

      • Firebase: БД для хранения данных о пользователях и их изображениях. А также хостинг сайта

      • GIT: Для создания репозитория с помощью которого можно отслеживать версии проекта

      • GITHUB: Хостинг репозитория

      API нейросети

      • Сайт использует внешний API для генерации изображений на основе текстовых запросов.

      • Обработка ответов API включает в себя отображение полученных изображений и их сохранение в базе данных.(ХРАНЕНИЕ ИЗОБРАЖЕНИЯ ДЕЙСТВУЕТ НЕ БОЛЕЕ 1 ЧАСА!)

      Управление состоянием

      • Использование JavaScript для управления состоянием приложения, обработки событий и манипуляции с DOM.

      • Динамическое обновление интерфейса при изменении данных (например, при добавлении нового изображения в галерею).
